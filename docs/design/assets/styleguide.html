Zone Input Pattern (Staff UI)

Staff UI does NOT use traditional text inputs. The entire zone (card/row) is the tap target.
Tapping a zone opens a modal for input, or expands inline for selection. No small text boxes.

Search Zone (tap to open modal)

Resting state: a tappable bar. Tap opens a search modal with real-time results.

Interaction Model Summary (Staff UI)

When to use modal vs inline expand.

Input Type Method Reason
Customer search Modal Result list needs full screen space
New customer name Modal Focused text input with keyboard
Selection (4-5 choices) Inline Chips appear in-place. Context stays visible
Memo / text note Inline Write while seeing tasks & customer info
Action (create record) Zone button Full-width tap target. Single action
